実運用環境で稼働できるTODO管理アプリケーションを

構築し提供できるようになるでしょう。

ただ、提供できるだけではなく、
有料のWEBサービスとして世に出すことができるように

インターネット上で見つかるソースコードや
レクチャーは、バージョンが古くて動かないケースが多いです。

このテクノロジーレクチャーを通して、
現在動くソースコードを実際に試した結果を
